<h1 align="center">Student List Manager</h1>
<h4 align="center">Spring Thymeleaf application</h4>

## Description

Small touch with thymeleaf.

Web were you can register a student, edit it, view all students and remove it.

## Technologies

- Java 17 (Minimum)
- Springboot
- Hibernate JPA
- Thymeleaf
- Lombok
- H2 database
